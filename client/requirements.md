## Packages
framer-motion | Page transitions and complex card flip animations
jspdf | PDF generation for the final report
html2canvas | Taking snapshots of HTML elements for PDF export
date-fns | Date and timer formatting
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge tailwind classes
lucide-react | Icons

## Notes
- Ensure tailwind.config.ts has the generated custom fonts 'Cinzel Decorative' and 'Outfit' configured if using specific font classes.
- API is expected to handle JSON payloads and use standard REST practices based on shared/routes.ts.
- The design is mobile-first (VK Mini App style). On desktop, it will appear as a centered mobile-proportioned app screen.
